# LibreOffice para Windows

Coloca aquí 'program/soffice.exe' (+ el resto del árbol de instalación de
LibreOffice de Windows) para que el PDF funcione sin instalación previa.
Ver scripts/fetch-libreoffice-win.md. Esta carpeta puede quedar vacía: el PDF
caerá al LibreOffice del sistema, o se desactiva si no hay ninguno.
